LAMP搭建与phpwind安装

发布时间 : 2022-02-11 08:40
阅读 :

    1. 一：部署LAMP环境
    2. YUM部署LAMP环境

LAMP

L –>Linux【操作系统】

A –>Apache【网页服务器】

M –>Mysql,MariaDB【数据库管理系统】

P –> PHP/Perl/Python【脚本语言】

LAMP是指一组通常一起使用来运行动态网站或者服务器的自由软件名称首字母缩写；

一：部署LAMP环境

YUM部署LAMP环境

  1. 快捷安装：
yum -y install httpd php php-mysql mysql mysql-php mysql-server
# mysql-php 和mysql-server 遇到过NO package情况，忽略即可
  1. 设置MySQL密码：
mysql -u root -p # 指定 root 用户登录 MySQL
set password for 'root'@'localhost'=password('passwd');  #修改密码
# set password for username @localhost = password(newpwd);
create database phpwind;

一键部署LAMP环境:

  1. 安装wget和git
yum -y install wget git
  1. git clone项目并赋予脚本执行权限
git clone https://github.com/teddysun/lamp.git
cd lamp
chmod 755 *.sh
  1. 安装
./lamp.sh

二：phpwind安装

必须关闭selinux和防火墙 ——重要！！！

selinux 0     # 临时关闭selinux

# 永久关闭selinux
# vim /etc/sysconfig/selinux
# 将SELINUX=enforcing 改为 SELINUX=disabled

systemctl stop firewalld  # 临时关闭防火墙

# 永久关闭防火墙
# systemctl disable firewalld

下载phpwind源码

# 将下载文件中的upload移到/var/www/html 下.
mv /path:phpwind/upload /var/www/html

修改权限：

chmod -R 777 /var/www/html/upload

Firefox http://127.0.0.1/upload  # 切回普通用户安装
