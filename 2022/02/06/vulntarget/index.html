windows 7

windows server 2012

  1. 安装redis

    https://link.zhihu.com/?target=https%3A//github.com/microsoftarchive/redis/releases/download/win-3.2.100/Redis-x64-3.2.100.msi
    • 将添加到环境变量勾选上一路默认

  2. 配置未授权

    1. 在redis安装目录下执行命令，关闭redis服务
    redis-server --service-stop

    1. 更改配置文件

      进入Redis的安装目录，然后修改windows.conf文件，第56行注释掉# bind 127.0.0.1，第75行把protected-mode yes改成protected-mode no 这样就可以模拟未授权访问的Redis靶机。

    2. 卸载Redis服务

      redis-server --service-uninstall

    3. 安装Redis服务

      redis-server --service-install redis.windows.conf

    4. 启动redis

      redis-server --service-start

    5. 将C:\Users\Administrator文件夹的组添加上Everyone并把权限改成完全控制

  3. web环境

    配置一个web环境，使用简单的phpstudy，设置了目录可写

    安装之后。可以访问phpstudy自带的php文件，但是访问重新写的php文件就是直接下载，没有解析到，这里需要修改apache配置文件

    去掉AddType application/x-httpd-php .php .phtml处注释
