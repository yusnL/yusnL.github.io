前言

PHP（PHP: Hypertext Preprocessor）即“超文本预处理器“，是在服务器端执行的一种解释型脚本语言，适用于Web开发并可嵌入HTML中。

本篇一直更新

php 代码与 html不结合时，建议不适用?>结尾（?> 有自带语句结束符 ‘;’ 的效果）

  • 加结尾：后续内容视为纯文本输出，可在php文件结尾添加html代码
  • 不加结尾：后续内容视为PHP代码直至没有代码，可防止html代码写入php文件

删除变量

用unset()来销毁变量，销毁的是变量名，变量值由PHP垃圾回收机制销毁

  • 注意在函数中unset全局、引用传递的变量，被销毁的只是函数内的局部变量

用isset() 用来检查变量是否被设置并是否为空（NULL）

php类型判断

类型判断函数：is_类型(变量) 返回值为bool型，不可用echo查看

$a = 123;
var_dump(is_int($a));

php文件包含

include/require # 包含文件
include_once/require_once # 包含一次文件
include 'filename' # 执行方式
文件包含方式

  1. 向上包含：在使用目标文件前包含，可以使用目标文件数据

    • 当前文件使用目标文件数据

  2. 向下包含：包含目标文件，在目标文件中写当前文件数据，目标文件识别不了当前文件数据，但当前文件 可运行目标文件中的当前文件数据

    • 在目标文件中写当前文件数据，目标文件可被当前文件调用执行
require和include区别：报错形式
  1. include报错不会阻断代码执行
  2. require报错不再执行require后面的代码
include和include_once区别
  1. include调用一次执行一次
  2. include_once和iclude一样，但会检测在自己之前有没有包含过该文件，若包含过则不执行

php 数据变化查询

mysqli_insert_id(mysql$link)、$pdo->lastInsertID():查询上条命令执行后自增ID的变化

mysqli_affected_rows(mysql$link)、$rs // $rs=$pdo->exec() :查询上条命令执行后受影响的记录数

$arr[]=1;

$arr[]=3;
// arr=[1,2,3]
