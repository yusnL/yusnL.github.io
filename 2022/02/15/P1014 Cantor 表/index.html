P1014 Cantor 表

题目描述

现代数学的著名证明之一是 Georg Cantor 证明了有理数是可枚举的。他是用下面这一张表来证明这一命题的：

1/1 , 1/2 , 1/3 , 1/4, 1/5, …

2/1, 2/2 , 2/3, 2/4, …

3/1 , 3/2, 3/3, …

4/1, 4/2, …

55/1, …

…

我们以 Z 字形给上表的每一项编号。第一项是 1/11/1，然后是 1/21/2，2/12/1，3/13/1，2/22/2，…

输入格式

整数NN（1 \leq N \leq 10^71≤N≤107）。

输出格式

表中的第 NN 项。

输入输出样例

输入

7

输出

1/4

思路

可以将三角形看成一条条斜线，设 i，j 两个数 i 为第几条斜线，j 为前 i 条斜线含有的数据总个数，且 i 条线包含 i 个数据，so直接循环硬干，当 j 大于给定的数时再减，最后看第 i 条从那个方向开始算起，看情况左右调换就OK

提交过程

一次提交 – AC

a = int(input())
b, c = 1, 1
while True:
    if b >= a:
        b -= a
        break
    c += 1
    b += c
print(f"{c-b}/{1+b}"if c % 2 == 0 else f"{1+b}/{c-b}")

二次提交

结构小优化

a = int(input())
b, c = 1, 1
while b < a:
    c += 1
    b += c
b -= a
print(f"{c-b}/{1+b}"if c % 2 == 0 else f"{1+b}/{c-b}")

exp 分析总结大佬思路 – 纯正高数思路，觉得大佬好厉害~~~发抖（{ { {(>_<)} } }）~~

对，歪头，斜过来看，第一行一个数，第两行两个数，第三行三个数。。。

好的，轻松看出等差数列，初中学过等差数列前 n 项和公式${(1 + n) × n \over 2}$

设所求数据为第 n 个，所在行数为 a ，第 a 行 n 最多到第${(1 + a) × a \over 2}$个，最少到${(a - 1) × a \over 2}$

得出：${(a - 1) × a \over 2}$ < n ≤ ${(1 + a) × a \over 2}$

到现在看到式子就有几分熟悉的感觉了

有一个数 x 唯一确定 n 的位置，且 a - 1 < x ≤ a

设：${(1 + x) × x \over 2} $= n ，由建立起来的函数的递增性可知：a = $\lceil x \rceil$

即：a = $\lceil {-1 + \sqrt{1 + 8 × n} \over 2} \rceil$

a 求出后，n 所在 a 的具体位置就好求了，可轻松得出 n 为第 a 行开始第 y 个数据

y = n - ${(a - 1) × a \over 2}$

可以看出每行左右两边相对应的数据只是 ‘/‘ 左右颠倒了而已

这样不管 y 是从左右那边开始都无所谓，可以轻松看出规律

so：

​ n = ${y \over a+1-y}^(-1^(a mod 2))$
