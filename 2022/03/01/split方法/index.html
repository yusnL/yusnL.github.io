split方法：通过特定的分隔符将字符串进行切片

用法

str.split(str="",num=string.count(str))

str: 默认所有空字符，包括空格、换行和制表等字符

num：默认为-1（负数）.代表所有

返回值：为分割后的字符串列表

1. 默认无和num=0时输出原字符串列表

"didimeni2".split()       # ['didimeni2']

"didimeni2".split("i",0)  # ['didimeni2']

2. 指定字符串”i”

"didimeni2".split("i")

# ['d', 'd', 'men', '2']

3. str不可省略

"di di men i  2".split(,2)

# SyntaxError: invalid syntax

修改建议如下：

"di di men i 2".split(" ",2)

# ['di', 'di', 'men i 2']

4. 注意若分割符位于字符串两端，则输出列表两端为””

"didimeni".split("i")    # ['d', 'd', 'men', '']      

"didimeni".split("d")    # ['', 'i', 'imeni']

"didimeni".split("i")[:-1]   # ['d', 'd', 'men']
